简介

Lua 是一门扩展式的编程语言，它被设计成使用数据描述工具来支持普通的过程式语言。 它还对面向对象编程、函数式编程、数据驱动编程提供了良好的支持。 Lua 能够作为一门强大的、轻量级的、嵌入式的脚本语言来供任何语言使用。 Lua 以库的形式实现，由 clean C(标准 C 与 C++ 的通用子集)写成。

作为一门扩展式语言，Lua 没有「main」 程序的概念：它只能工作在嵌入的宿主委托(host client)中，宿主委托也叫「embedding program」或干脆简称为「host」。 宿主程序能够通过调用函数来执行一段 Lua 代码，能够读写 Lua 变量，还能够注册 C 函数来让 Lua 代码调用。 通过使用 C 函数，极大的扩展了 Lua 使其能够处理不同领域。这样就能够定制出共享同一个句法框架的不同语言。 Lua 的官方发布版包含了一个名为 lua 的示例宿主程序，它使用 Lua 库来提供一个完整独立的 Lua 解释器，用于交互式或批处理。

Lua 是自由软件，它所使用的许可证规定了它不会提供任何担保。本手册描述的实现可以在 Lua 的官方网站 www.lua.org 找到。

与其他任何参考手册一样，这份文档在有些地方会比较枯燥。关于 Lua 设计背后的决策讨论，请参见 Lua 官网上提供的技术论文。要了解更多 Lua 编程细节，请阅读 Roberto 的「Programming in Lua」。
